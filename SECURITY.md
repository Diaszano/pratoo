# Política de Segurança

## Reportando uma vulnerabilidade

Valorizamos a segurança do Pratoo. Se você encontrou uma vulnerabilidade de segurança, por favor reporte-a de forma responsável.

**Por favor, NÃO abra uma issue pública para vulnerabilidades de segurança.**

### Como reportar

Envie um email para o mantenedor diretamente (disponível no perfil do GitHub) com os seguintes detalhes:

- Descrição do problema
- Passos para reproduzir
- Impacto potencial
- Sugestão de correção (se houver)

### O que esperar

- Confirmação de recebimento em até 48 horas
- Atualizações a cada 7 dias sobre o progresso
- Atribuição de CVE (se aplicável)
- Reconhecimento público após a correção (se desejado)

### Escopo

Aplicam-se a este repositório:

- Código fonte no branch `main`
- Dependências gerenciadas pelo Gradle
- Scripts de CI/CD

**Fora do escopo:**

- Instâncias não oficiais ou forks do projeto
- Dispositivos com root/jailbreak
- Versões modificadas do app

## Boas práticas

- Mantenha o app atualizado via as releases oficiais
- Não modifique o APK após o download
- Use apenas fontes oficiais (GitHub Releases ou Google Play, quando disponível)
